# Fibonacci-Nim-Game
A command line game based of the Fibonacci Nim 

**Main features**

Input error checking

Enforces rules of Fibonacci Nim game (see wikepedia [https://en.wikipedia.org/wiki/Fibonacci_nim#Strategy])
